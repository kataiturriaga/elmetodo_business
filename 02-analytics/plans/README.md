## Cómo retomar este trabajo

Este folder (`02-analytics/plans/`) guarda documentación y planes de analytics **en formato Markdown**.

### Plan “editable” con botón Build en Cursor

Si quieres ver el botón **Build** de Cursor (planes), abre el plan desde:

- `.cursor/plans/dashboard_kpis_looker.plan.md`

Ese archivo está dentro del repo para que puedas versionarlo, y Cursor lo reconoce como *plan file*.

### Copia legible en analytics

También existe una copia del mismo plan aquí:

- `02-analytics/plans/dashboard_kpis_looker.plan.md`

Úsala como referencia rápida desde la carpeta de analytics.
